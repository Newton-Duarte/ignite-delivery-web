# Ignite Delivery - Web

A project to consume the API from the ignite delivery prisma course

## Backend project for this web app

https://github.com/Newton-Duarte/ignite-delivery-prisma

## How to run this project

#### Install dependencies

`$ npm install`

or

`$ yarn`

#### Start the server

`$ npm run dev`

or

`$ yarn dev`

#### Project URL

`http://localhost:3000`
